# Financial Transition Plan — Single-Income Household

*Tracks the plan, the numbers, and the decisions behind moving the household onto Marty's income alone.*

---

## 1. Context & The Goal

- Nick's job is at meaningful risk of elimination as early as August 2026. He's nearly 60.
- The goal: make it possible for Nick to be done working — not forced into a new job at this stage of his life. He may pick up other work later, but the plan must not depend on it.
- This is a one-income transition plan: prove out living on Marty's salary alone, while building the cushion and clearing the debt that make it sustainable.

## 2. The Transition Plan

**Current state (per paycheck — biweekly, sometimes 3 paychecks in a month):**
- Marty: $1,500 → checking / $2,224 → savings ($3,724 total) — confirmed against actual deposit data 6/24/26
- Nick: $2,600 → checking / $500 → savings ($3,100 total) — confirmed against actual deposit data 6/24/26
- Mostly cover bills on combined checking deposits, but self-described "frivolous" spending means savings gets dipped into often.
- Both work for Trinity Health, biweekly, pay dates offset by ~2 days from each other — this caused real confusion when reading raw bank deposit data this session (see Section 5).

**Target state (in progress — needs revisiting, see note below):**
- Marty: $3,700 → checking / balance → savings
- Nick: full paycheck → savings
- Test: whether the household can actually run on $3,700/paycheck from checking alone.
- Note: biweekly pay means some months have 3 paychecks per person — those months are a built-in extra savings/cushion-building window worth flagging when they land.
- **Open flag (6/24/26):** the $3,700 checking target predates having a real floor number. The actual floor (Section 4) is $5,547.19/month, and Marty's biweekly check averages $3,730.96 — close to $3,700 but the floor needs to be checked against monthly cash timing (3-paycheck months vs. 2-paycheck months), not just assumed to clear. Revisit this number once discretionary categories are reviewed.

**Prerequisites identified:**
- Pay off existing credit card balances — in progress, see Section 3
- Eliminate Nick's $500/mo Jeep payment — not yet addressed
- **New prerequisite found this session:** Marty's own vehicle loan ($950/mo) currently auto-drafts from savings. That only works because Nick's paycheck keeps savings fed. Once Nick stops earning, this payment has to be funded by checking/Marty's income directly — it's now included in the Section 4 floor.

## 3. Debt Payoff Plan
